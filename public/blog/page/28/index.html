Thunderbird 1.5 Arrives

Jan 12th, 2006 Comments

Thunderbird 1.5 has been released. It’s ready to download, so get going and download it already.

Here’s what’s new in Thunderbird 1.5:

    * Automated update to streamline product upgrades. * Sort address autocomplete results by how often you send e-mail to each recipient. * Spell check as you type. * Saved Search Folders can now search across multiple accounts. * Built in phishing detector to help protect users against email scams. * Podcasting and other RSS Improvements. * Deleting attachments from messages. * Integration with server side spam filtering. * Reply and forward actions for message filters. * Kerberos Authentication. * Auto save as draft for mail composition. * Message aging. * Filters for Global Inbox. * Improvements to product usability including redesigned options interface, and SMTP server management. * Many security enhancements.
