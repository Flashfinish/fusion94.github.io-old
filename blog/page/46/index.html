SourceForge & IA-64

May 26th, 2000
  • Open Source
  • SourceForge
Comments

We’re pleased to announce that SourceForge now has IA-64 (Itanium) machines available to use in the compile farm.

There are a few requirements such as a NDA, no benchmarking (or at least publicy stating the numbers), the project must be open source, and access has to go through an approval process. That is the overview that we’ve come up with and we hope that this will go a long way in finding out any IA-64 issues that your code may have. This will also be good for developers in checking the IA-64 Linux Kernel out in real world conditions.

SourceForge Launched

Nov 17th, 1999
  • Cold Storage
  • Linux
  • SourceForge
  • VA Research
Comments

So what was once “Cold Storage” is now SourceForge and is now available for public consumption. So what is SourceForge? Well it’s a web resource for Open Source projects and organizations. We’re providing CVS, mailing lists, public forums, the ability to poll visitors and project members, bug tracking and anonymous FTP for all open source projects to anyone completely free of charge.

I encourage anyone that’s involved in an open source project to stop by and try it out.

Big thanks go out to Larry Augustin, Tim Perdue, Drew Streib, and Uriah Welcome for making this happen.
